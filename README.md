# The Vault — CPA Grip Landing Pages

A premium CPA (Cost Per Action) marketing landing page system designed to convert visitors into offer completions. Built for the CPA Grip network.

## Overview

**The Vault** is a high-converting content locking system that offers visitors premium free resources (ebooks, videos, tools) in exchange for completing CPA offers. When a user completes an offer, they "unlock" the promised content.

### How It Works

1. **Landing Page** — Visitors arrive at a compelling offer page showcasing 20+ premium resources
2. **User Action** — Visitor completes a CPA offer (email submit, survey, app install, etc.)
3. **Unlock** — Upon offer completion, the user gains access to the locked content

## Project Structure

```
├── index.html              # Main landing page
├── offers.html             # Offer wall with available CPA offers
├── unlock-ebooks.html      # Ebook library (unlocked after offer)
├── unlock-videos.html      # Video library (unlocked after offer)
├── cpa-grip-content-locker # Content locking script
├── cpa-grip-offer-wall     # Offer wall integration
└── cpa-grip-video-locker   # Video locker integration
```

## Features

- **Premium Design** — Dark, luxurious aesthetic with gold accents

- **Content Locking** — Gates content behind CPA offers

## Setup

1. Upload files to your hosting server
2. Replace placeholder CPA Grip offer links with your actual campaign URLs
3. Configure the content locker scripts with your CPA Grip publisher ID
4. Test the flow to ensure offers track correctly

## CPA Grip Integration

These pages are designed to work with [CPA Grip](https://cpagrip.com) — a leading CPA affiliate network. You'll need to:

- Sign up as a publisher at CPA Grip
- Create your campaigns
- Replace the `offer_url` values in the scripts with your unique tracking links
